# 艺培续费增长助手 MVP

面向艺术培训机构的续费增长后台原型，支持登录、学员录入、流失风险评分、家长沟通话术生成，以及从本地知识库文件中自动识别学员信息。

## 功能

- 后台登录：默认账号 `admin`，密码 `123456`
- 学员手动录入：课程、老师、课时、缺勤、家长互动、续费金额、成长证据
- 风险看板：自动计算高风险、快课消、家长沉默学员
- 续费话术：按温和、直接、高客单三种语气生成沟通内容
- AI 录入：扫描 `knowledge_base` 文件夹里的 `.txt`、`.md`、`.json`、`.csv` 文件，抽取候选学员并确认导入

## 本地运行

```bash
npm start
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 配置

可通过环境变量覆盖默认配置：

```bash
PORT=4173
ADMIN_USER=admin
ADMIN_PASS=123456
```

## 知识库文件

把学员记录放入 `knowledge_base` 文件夹。示例：

```text
学员姓名：孙若晴
课程：少儿创意美术进阶
负责老师：唐老师
剩余课时：2
预计课消天数：5
缺勤率：12
家长回复次数：0
作业缺交次数：1
续费金额：4980
最近联系：9天前
成长证据：构图开始有主次，能主动安排主体和背景关系。
```

## 部署说明

这是一个 Node 后台项目，不是纯静态网站。GitHub Pages 只能托管静态页面，不能运行登录、API、本地 JSON 写入和知识库扫描。

推荐部署方式：

- GitHub：托管代码仓库
- Render、Railway、Fly.io、VPS：运行 Node 服务

生产环境还需要替换本地 JSON 存储，改为数据库，并把默认密码改成强密码。
